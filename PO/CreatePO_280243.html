===========================
WORKFLOW INFORMATION
===========================


To automatically create a PO document using workflow, there are a series of steps that need to complete successfully as follows:

  • Requisition approval process should successfully approve the requisiiton. Verify the activity 'APPROVE_REQUISITION' is completed normal.

  • Requisition approval process launches a second process for PO creation. Verify the activity 'LAUNCH_CREATE_PO_WF' is successfully completed.

    • A new WF process with Item type CREATEPO and root activity name OVERALL_AUTOCREATE_PROCESS should be created. Verify the WF was successfully launched.

    • The CREATEPO should create 2 new WF process

      • One LAUNCH_REQ_LINE_PROCESSING process for each requisition line

      • After all lines where processed, a LAUNCH_DOC_CREATION_APPROVAL process. Verify the activity CREATE_DOC completes in status CREATE_OK

      • note that if CREATE_DOC is completed ok, this process also submits a new process to approve the PO document. Activity name is LAUNCH_PO_APPROVAL
